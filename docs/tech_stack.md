# Technology Stack - Hierar-Do

## 1. Backend
- **Framework**: FastAPI (Asynchronous Python)
- **Agent Orchestration**: LangGraph (for stateful multi-agent workflows)
- **LLM**: Claude 3.5 Sonnet / GPT-4o

## 2. Database
- **Main DB**: PostgreSQL
- **Vector Search**: pgvector (for long-term memory & context retrieval)

## 3. Tooling
- **Package Manager**: Poetry or Pipenv
- **Environment**: Docker (for consistent dev environment)
- **API Integration**: Google Calendar API (future)
